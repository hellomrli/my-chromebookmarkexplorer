# Changelog

## [0.2.1] - 2026-07-06

### Fixed
- 修复合并冲突，整合远程新增功能
- 确保所有工具栏功能正常工作（另存为、编辑网址、查重、删除/移动异常链接）

## [0.2.0] - 2026-07-06

### Added
- 应用图标和文件夹/书签图标
- 右侧列表增加勾选框，支持批量选择操作
- 工具栏按钮增加图标显示
- 搜索框增加占位符提示
- 表格交替行颜色显示
- 右键菜单增加全选/取消全选功能

### Changed
- 优化界面布局：增大默认窗口尺寸到 1200x760
- 优化列宽度分配：勾选框固定宽度 32px
- 优化工具栏样式：图标在文字旁边显示
- 优化状态栏样式：增加顶部边框
- 左侧文件夹树增加文件夹图标

### Fixed
- 修复列索引偏移问题（增加勾选框列后）
- 优化批量操作逻辑：优先处理勾选项，其次处理选中项

## [0.1.0] - 2026-07-05

### Added
- 初始版本发布
- Chrome Profile 自动识别
- 文件夹树和书签列表视图
- 新建、重命名、删除、移动功能
- 网址测活功能
- 自动备份和原子写入
