# Changelog

## [0.2.2] - 2026-07-13

### Changed
- 网址测活改用浏览器式轻量 GET，请求收到最终响应头后即停止正文下载，避免 HEAD 兼容性误判
- 401/403/429/451 改为“需要登录/访问受限/访问限流”，服务器可达时不再计作死链
- 默认使用系统代理，增加每主机并发限制、15 秒超时、瞬时故障重试、协议回退和 HTTP/1.1 回退
- 无协议网址优先 HTTPS，旧 HTTP 连接失败时尝试 HTTPS；非 HTTP 书签改为跳过而非失败
- 每次检测会清除当前范围的旧结果，实时测活默认关闭结果缓存；删除异常链接时只默认勾选较明确的失效结果

### Added
- 测活结果增加状态类型、可达性、确定失效判断、尝试次数和详细提示
- 增加本地 HTTP 测试，覆盖 GET/HEAD 差异、访问限制、404、跳转、重试和每主机并发限制

## [0.2.1] - 2026-07-06

### Fixed
- 修复合并冲突，整合远程新增功能

## [0.2.0] - 2026-07-06

### Added

- 右键菜单增加全选/取消全选功能

### Changed
- 优化界面布局：增大默认窗口尺寸到 1200x760
- 优化列宽度分配：勾选框固定宽度 32px
- 优化工具栏样式：图标在文字旁边显示
- 优化状态栏样式：增加顶部边框
- 左侧文件夹树增加文件夹图标

### Fixed
- 修复列索引偏移问题（增加勾选框列后）
- 优化批量操作逻辑：优先处理勾选项，其次处理选中项
